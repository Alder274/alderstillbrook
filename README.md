## Replace later
Add translated covers using these filenames:

- `assets/covers/cover-nl.webp`
- `assets/covers/cover-en.webp`
- `assets/covers/cover-es.webp`
- `assets/covers/cover-pt.webp`
- `assets/covers/cover-de.webp`
- `assets/covers/cover-fr.webp`
- `assets/covers/cover-it.webp`

If a translated cover is missing, the script falls back to the English cover.
